These technologies are related to web pages or websit developement. It all started with WWW(world Wide Web) at CERN by Tim Berners Lee. Later He implemented first browser findly markup language .GML Generalised Mark-Up Language but later updated as HTML HyperText Markup Language.It is the standard language that works in all browser. The current version of HTML is HTML5. Images can be inserted into web pages using img tag. It will take attribute such src and alt. Modern html supports responsive images such as displaying one image for desktop and another image for mobile. It is impossible using picture tag.
